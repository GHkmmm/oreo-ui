# 开始使用



请先[安装](#/docs/install)本组件库

然后在你的代码中写入下面的代码

```
import { OrButton, OrTabs, OrSwitch, OrDialog } from "oreo-ui"
```

就可以使用我提供的组件了



## Vue 单文件组件

代码示例

```
<template>
  <div>
    <or-button>按钮</or-button>
  </div>
</template>

<script>
import { OrButton, OrTabs, OrSwitch, OrDialog } from "oreo-ui"
export default {
  components: {OrButton}
}
</script>
```
